Procedure

  1. To obtain the transfer function of AC servo motor, user should use general torque equations of AC servo motor.
  2. For the calculation of angular velocity of AC servo motor, the user should enter values for Moment of inertia, Viscous frictional coefficient and slope of the speed and torque characteristics.
  3. For the calculation of angular displacement, the user should entervalues for slope of the speed and torque.
  4. To obtain overall block diagram of AC servomotor, user should use transfer function and angular velocity of AC servo motor.
  5. To obtain the characteristics of AC servo motor, the user should tabulate speed & input voltage; Speed of the motor changes with respect to variations of input voltage.
