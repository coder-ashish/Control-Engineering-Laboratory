Aim

The user interacts with the virtual lab through the web page which contains the transfer function of AC servomotor to obtain the angular velocity. The user is proposed to have information on the various equations required to analyze the speed â€“ torque relation. This transfer function includes various known parameters like Moment of inertia, Viscous frictional coefficient and slope of the speed and torque characteristics to obtain the angular velocity. The unknown angular velocity obtained by giving the various above input parameters. With help of transfer functions and angular velocity, the overall block diagram of AC servo motor is constructed. From the block diagram used can tabulate the values of speed with respect to variation of input voltage. User may also use the help option to have a better clarity on the doubts that arise during the conduction of experiments. After completing this experiment, user will get idea about behavior AC servomotor based speed-voltage characteristics. The user can able to understand the concept of how angular displacement changes with respect to speed and torque slope values. After gaining knowledge from this experiment, the user can do the analyze on features of AC servomotor like steady state stability, linear relationship between speed and electrical signal and wide range of speed control.
